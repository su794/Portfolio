Tesla

Responsive Tesla Webpage

The goal was to produce one page of a responsive web design site with 5 sections. Theme was subject (but real content) and I chose Tesla company to apply jQuery plugins. This was for Final project of Animation and Mobile Web. Upon adjusting screen size, contents such images, buttons will be different in every breakpoint.

Technical Info:

HTML, CSS, JavaScript, Adobe Photoshop, jQuery plugin (Flickity, ScrollMagic, Accordion, MixitUp)

PROCESS

This was given as a final project of Animation and Mobile Web Class. We were supposed to use our HTML/CSS and JavaScript/jQuery skills with a responsive web design. Students chose a website theme with real content. I went with Tesla.

01

Hamburger menu and video background

In a mobile device screen size, there is a hamburger menu, I designed and created a shape with a html and css skills and javascript to make it work. When a hamburger menu is opened, scroll is locked and menu is clicked, it goes to the place I chose smoothly.

02

Breakpoints

I used 4 breakpoints-576px, 768px, 992px, 1200px-but started code from 320px, smallest screen size(mobile). I usually complete 320px size first and go to next breakpoint. In this process, size of buttons, fonts, images or contents gets bigger and placed in a different way.

Background images were resized in Photoshop based on a breakpoint.

03

5 sections and jQuery plugins

1: Banner section-Background image with two cta buttons and texts over it. under the image, there are three columns with images indicating tesla's service and value.

2: About-ScrollMagic Plguin. while scrolling down, an image and items of the list come in and out when scrolling up.

3: Shop-MixitUp, when clicking an item category, items are sorted accordingly. It shows a item in a row in smaller screen size and becomes 2 and 3 with a bigger size of images. For the category labels, when it's clicked, I used CSS skills to make it look like it's dented.

4: FAQ-Accordion, when we click other titles, the current one closes. And the direction of the arrow will be changed.

5: Contact form with HTML and CSS. from 1200px, hover effect is added.
